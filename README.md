# Sales-Forecasting-Project
Sales forecasting project using Python and ARIMA to predict future sales from historical time-series data. The workflow includes data preprocessing, trend visualization, model training, and forecast evaluation. This project demonstrates practical time-series analysis for business decision-making in retail and e-commerce scenarios
